# Reachy Mini Scratch サンプルプロジェクト

TurboWarpで以下のサンプルプロジェクトを作成してみましょう！

## 🎯 サンプル1: 基本的な動き

**目的**: ロボットの基本的な動きをテストする

**ブロック構成**:
```
緑の旗が押されたとき
  ロボットを起こす
  [2]秒待つ
  頭を [上] に動かす
  [1]秒待つ
  頭を [下] に動かす
  [1]秒待つ
  頭を [左] に動かす
  [1]秒待つ
  頭を [右] に動かす
  [1]秒待つ
  頭を正面に戻す
  [2]秒待つ
  ロボットを寝かせる
```

---

## 🤖 サンプル2: アンテナの動き

**目的**: アンテナを使った感情表現

**ブロック構成**:
```
緑の旗が押されたとき
  [10]回繰り返す
    アンテナを [20]° にする
    [0.3]秒待つ
    アンテナを [-20]° にする
    [0.3]秒待つ
  終わり
  アンテナを [0]° にする
```

---

## 🎭 サンプル3: インタラクティブ制御

**目的**: キーボードでロボットを制御

**ブロック構成**:
```
緑の旗が押されたとき
  ロボットを起こす

[上向き矢印▲]キーが押されたとき
  頭を [上] に動かす

[下向き矢印▼]キーが押されたとき
  頭を [下] に動かす

[左向き矢印◀]キーが押されたとき
  頭を [左] に動かす

[右向き矢印▶]キーが押されたとき
  頭を [右] に動かす

[スペース]キーが押されたとき
  頭を正面に戻す
```

---

## 📊 サンプル4: センサー値の表示

**目的**: ロボットの現在の状態を表示

**ブロック構成**:
```
緑の旗が押されたとき
  ずっと
    [pitch: ]と[頭のpitch角度]と言う
    [0.5]秒待つ
    [yaw: ]と[頭のyaw角度]と言う
    [0.5]秒待つ
  終わり
```

---

## 🎵 サンプル5: ダンス

**目的**: 複雑な動きの組み合わせ

**ブロック構成**:
```
緑の旗が押されたとき
  ロボットを起こす
  [2]秒待つ
  [4]回繰り返す
    頭を [左上] に動かす
    アンテナを [30]° にする
    [0.5]秒待つ
    頭を [右上] に動かす
    アンテナを [-30]° にする
    [0.5]秒待つ
  終わり
  頭を正面に戻す
  アンテナを [0]° にする
  [2]秒待つ
  ロボットを寝かせる
```

---

## 💡 追加のアイデア

### カスタム角度での制御
```
頭を動かす pitch:[10] yaw:[15] roll:[0] 時間:[2.0]秒
```
このブロックを使うと、より細かい制御が可能です！

### 条件分岐の例
```
もし [頭のpitch角度] > [10] なら
  頭を [下] に動かす
でなければ
  頭を [上] に動かす
終わり
```

# 🚀 クイックスタートガイド

Reachy Mini Scratch拡張機能を5分で始める方法！

## ステップ1: Reachy Mini daemonを起動 (ターミナル1)

### シミュレーションの場合:
```bash
cd ~/projects/reachy_mini
reachy-mini-daemon --sim
```

### 実機の場合:
```bash
cd ~/projects/reachy_mini
reachy-mini-daemon
```

✅ `http://localhost:8000` でダッシュボードが起動したら成功！

---

## ステップ2: 拡張機能サーバーを起動 (ターミナル2)

```bash
cd ~/projects/scratch-reachy-mini
python3 serve_extension.py
```

✅ 以下のように表示されたら成功！
```
🚀 Server starting on http://localhost:3000
```

---

## ステップ3: TurboWarpで拡張機能を読み込む

1. **TurboWarpを起動**
   ```bash
   turbowarp
   ```

2. **拡張機能を追加**
   - 左下の「**拡張機能を追加**」ボタンをクリック

3. **カスタム拡張機能を選択**
   - 一番下までスクロール
   - 「**カスタム拡張機能**」をクリック

4. **URLを入力**
   ```
   http://localhost:3000/reachy-mini-extension.js
   ```

5. **読み込み完了！**
   - ブロックパレットに「**Reachy Mini**」カテゴリが表示されます

---

## ステップ4: 最初のプログラムを作る

以下のブロックを組み立ててみましょう：

```
🏁 緑の旗が押されたとき
   ロボットを起こす
   [2]秒待つ
   頭を [上] に動かす
   [1]秒待つ
   頭を [下] に動かす
   [1]秒待つ
   頭を正面に戻す
   [2]秒待つ
   ロボットを寝かせる
```

---

## ステップ5: 実行！

緑の旗🏁をクリックして、ロボットが動くのを見てみましょう！

---

## 🎯 次にやってみること

### 1. キーボード制御
```
[上向き矢印▲]キーが押されたとき
   頭を [上] に動かす
```
同様に他の矢印キーも設定してみましょう！

### 2. アンテナを動かす
```
[10]回繰り返す
   アンテナを [30]° にする
   [0.3]秒待つ
   アンテナを [-30]° にする
   [0.3]秒待つ
終わり
```

### 3. センサー値を表示
```
ずっと
   [pitch: ]と[頭のpitch角度]と言う
   [1]秒待つ
終わり
```

---

## 💡 ヒント

- **各動作には時間がかかる**: 動作の間に`待つ`ブロックを入れましょう
- **角度の範囲**:
  - 頭: pitch ±30°、yaw ±40°、roll ±30° くらいが安全
  - アンテナ: ±45° くらいが安全
- **デバッグ**: F12キーでブラウザのコンソールを開いてエラーを確認できます

---

## ❓ 問題が起きたら

### ロボットが動かない
1. daemon（localhost:8000）が起動しているか確認
2. 拡張機能サーバー（localhost:3000）が起動しているか確認
3. ブラウザのコンソール（F12）でエラーを確認

### 拡張機能が読み込めない
```bash
# サーバーが動いているか確認
curl http://localhost:3000/reachy-mini-extension.js
```

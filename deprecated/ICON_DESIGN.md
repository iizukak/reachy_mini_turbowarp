# Reachy Mini Scratch Extension - アイコンデザイン

## 🎨 アイコンの特徴

Reachy Miniの特徴的な外観を反映したカスタムアイコンを作成しました。

### デザインコンセプト

**Reachy Miniの特徴を表現**:
- ✅ 2本の特徴的なアンテナ（先端の球体なし、実際のデザインに忠実）
- ✅ 大きな表情豊かな目（左右を横ラインでつなぐ）
- ✅ 横長の長方形の顔（実際のReachy Miniに近い形状）
- ✅ 口なし（実際のReachy Miniには口が存在しない）
- ✅ 親しみやすいデザイン

### カラースキーム

| 要素 | カラー | 説明 |
|------|--------|------|
| **背景円** | `#4C97FF` | Scratchの標準ブルー |
| **頭部（長方形）** | `#FFFFFF` | 白（清潔感） |
| **目** | `#4C97FF` | メインカラーで統一感 |
| **アンテナ** | `#FFFFFF` | 白（シンプル、先端の球体なし） |
| **ハイライト** | `#FFFFFF` (半透明) | 立体感の演出 |

## 📐 技術仕様

### SVGファイル

```xml
<!-- reachy-mini-icon.svg -->
<svg width="60" height="60" viewBox="0 0 60 60" xmlns="http://www.w3.org/2000/svg">
  <!-- 背景円 -->
  <circle cx="30" cy="30" r="28" fill="#4C97FF"/>

  <!-- ロボットの頭部（横長の長方形） -->
  <rect x="16" y="26" width="28" height="18" rx="4" fill="white"/>

  <!-- 左目（楕円形で表情豊か） -->
  <ellipse cx="24" cy="35" rx="3.5" ry="4.5" fill="#4C97FF"/>
  <ellipse cx="24" cy="34" rx="1.8" ry="2.2" fill="white" opacity="0.6"/>

  <!-- 右目 -->
  <ellipse cx="36" cy="35" rx="3.5" ry="4.5" fill="#4C97FF"/>
  <ellipse cx="36" cy="34" rx="1.8" ry="2.2" fill="white" opacity="0.6"/>

  <!-- 目をつなぐ横のライン -->
  <line x1="27.5" y1="35" x2="32.5" y2="35" stroke="#4C97FF" stroke-width="2.5" stroke-linecap="round"/>

  <!-- 左アンテナ（ラインのみ、球体なし） -->
  <line x1="22" y1="26" x2="16" y2="12" stroke="white" stroke-width="3" stroke-linecap="round"/>

  <!-- 右アンテナ（ラインのみ、球体なし） -->
  <line x1="38" y1="26" x2="44" y2="12" stroke="white" stroke-width="3" stroke-linecap="round"/>

  <!-- ボディの示唆 -->
  <rect x="20" y="44" width="20" height="4" rx="2" fill="white" opacity="0.7"/>
</svg>
```

### サイズ

- **viewBox**: `0 0 60 60`
- **実際のレンダリング**: 自動調整（SVGの利点）
- **ブロックアイコン**: 約20x20px
- **メニューアイコン**: 約60x60px

## 🎨 カスタマイズ方法

### アイコンを変更する

1. **SVGファイルを編集**:
   ```bash
   nano reachy-mini-icon.svg
   ```

2. **Base64エンコード**:
   ```bash
   base64 -w 0 reachy-mini-icon.svg
   ```

3. **コードに反映**:
   ```javascript
   const iconURI = 'data:image/svg+xml;base64,<エンコード結果>';
   ```

### デザインのヒント

#### 色を変える
```xml
<!-- 青系から緑系に変更 -->
<circle cx="30" cy="30" r="28" fill="#4CD964"/>  <!-- 元: #4C97FF -->
```

#### アンテナの形を変える
```xml
<!-- 直線から曲線に -->
<path d="M 22 20 Q 18 15 16 10" stroke="white" stroke-width="3"/>
```

#### 目の形を変える
```xml
<!-- 楕円から円に -->
<circle cx="25" cy="30" r="3" fill="#4C97FF"/>
```

## 📝 デザインガイドライン

### 推奨事項

1. **シンプルさ**: 小さく表示されても認識できるように
2. **コントラスト**: 背景色（白/灰色）でも見やすく
3. **ブランドカラー**: Scratchブルー（#4C97FF）を基調に
4. **SVG使用**: ベクター形式で拡大縮小に対応
5. **ファイルサイズ**: 軽量に保つ（現在: 約2KB）

### 避けるべき事項

- ❌ 細かすぎるディテール（小さく表示される）
- ❌ 多色使い（3〜4色まで）
- ❌ 複雑なグラデーション
- ❌ テキストの使用（読めない）
- ❌ 大きすぎるファイルサイズ
